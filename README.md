## Control keys

| Keys            | Action                  |
| --------------- | ----------------------- |
| Space           | stop/resume simulation  |
| Q, W            | set sail angle          |
| Right, Left, Up | set rudder angle        |
| Backspace       | reverse wind direction  |

| R               | reset simulation        |
